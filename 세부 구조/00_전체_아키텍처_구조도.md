# ToThereOn 전체 시스템 아키텍처

## 핵심 파이프라인 흐름도

```
┌─────────────────────────────────────────────────────────────────────┐
│                    STAGE 1: PET REGISTRATION                        │
│  /dashboard/pets/new → PetRegistrationForm                         │
│  ├─ pet_limit 체크 (users.pet_limit or max_pets_allowed)           │
│  ├─ 사진 업로드 (Supabase Storage, 1-5장)                          │

│  ├─ 5 Phase 진행 (The Door → Living Room → Quiet Room → Letter → Light) │
│  ├─ 5초마다 자동저장 → deep_remembrance_responses                   │
│  └─ 완료 시 → POST /api/deep-remembrance/complete                  │

                           │

│  │   GPT-4o Vision → pets.visual_description      │ (병렬, fire&forget)│

│  │                         breedRef, ageProfile)   │                 │

│  │   ├─ Breed Reference                            │                 │
│  │   ├─ Big Five Traits                            │                 │
│  │   ├─ Speech Style                               │                 │

│  │   ├─ Unique Traits                              │                 │
│  │   ├─ Relationship Bond                          │                 │
│  │   ├─ ⭐ HEALING MISSION (CORE OBJECTIVE)         │                 │
│  │   ├─ Letter Voice Style (Q19)                   │                 │
│  │   └─ Emotional Readiness (Q21)                  │                 │
│  └────────────────────────────────────────────────┘                  │
│                           │                                         │
│                           ▼                                         │
│  ┌─ [D] Database Save ───────────────────────────┐                  │
```

## AI 모델 사용 맵

| 단계 | 모델 | 용도 | Temperature |
|------|------|------|------------|
| Visual DNA | GPT-4o | 사진 → 외형 묘사 | 0.3 |
| NLP 분석 (주관식) | Claude Sonnet 4.6 | 설문 텍스트 → 성격/감정 추출 | 0.2-0.3 |
| Healing Mission 추출 | Claude Sonnet 4.6 | Q17+Q18 → core_desire, guilt | 0.2 |
| 피드 이벤트 생성 | Claude Sonnet 4.6 | 일일 상태 업데이트 | 0.95 |
| 편지 답장 생성 | Claude Opus 4.6 | 감정적 편지 작성 | 0.85-1.0 |
| QA 자동검수 | Claude Haiku 4.5 | 세계관/안전/페르소나 검증 | 0 |
| RAG 임베딩 | OpenAI text-embedding-3-small | 편지 벡터 저장/검색 | - |

## 핵심 트리거 플래그

| 플래그 | 위치 | 설정 시점 | 읽는 곳 |
|--------|------|-----------|---------|
| `persona_generated` | pets 테이블 | 페르소나 저장 후 | CRON, PetCard UI, WorldState |
| `survey_version` | deep_remembrance_responses | 설문 시작 시 | complete API → analyzer 분기 |
| `healing_mission` | pet_personas | 분석 완료 시 | reply-generator 시스템 프롬프트 |
| `analysis_version` | pet_personas | 분석 완료 시 | 디버깅/추적 |
| `is_active` + `version` | survey_questions | 마이그레이션 | 설문 UI 질문 로드 |

## 시간 시스템 요약

| 상수 | 값 | 의미 |
|------|---|------|
| TIME_RATIO | 3 | 지구 3일 = ToThereOn 1일 |
| 편지 배달 (crossing) | 72h | 보낸 후 72시간 뒤 도착 |
| 편지 읽기 (reading) | 96h | 도착 후 24시간 뒤 읽기 시작 |
| 답장 쓰기 (writing) | 120h | 읽은 후 24시간 뒤 쓰기 |
| 답장 공개 (visible) | 168h (7일) | 보낸 후 7일 뒤 사용자에게 노출 |
| 피드 주기 (Free) | 7 Earth days | 주 1회 |
| 피드 주기 (Basic) | 3 Earth days | 주 2회 |

## 파일 의존성 맵

```
lib/
├─ analysis/
│  ├─ master-analyzer.ts      ← 핵심 12-step 파이프라인
│  │  ├─ uses: nlp-analyzer, multiple-choice-handler, age-analyzer, breed-reference
│  │  └─ output: DeepPersonaAnalysis
│  ├─ nlp-analyzer.ts         ← Claude API 호출, healing_mission 추출
│  ├─ multiple-choice-handler ← Q04/Q06/Q12/Q13 패턴분석
│  ├─ prompt-generator.ts     ← 분석 → system prompt 조립
│  ├─ age-analyzer.ts         ← 나이/생애단계 분석
│  └─ breed-reference.ts      ← 품종 기준 데이터 조회
├─ worldview-constants.ts     ← 세계관 전체 상수 (존/NPC/규칙)
├─ zone-manager.ts            ← 존 진행, NPC 선택, 위치 선택
├─ event-generator.ts         ← 피드 이벤트 LLM 생성
├─ prompt-builder.ts          ← 피드용 유저 프롬프트 조립
├─ reply-generator.ts         ← 편지 답장 LLM 생성 + QA 루프
├─ time-engine.ts             ← ToThereOn 시간 계산
├─ time-constants.ts          ← 시간비율, 학습단계, 편지 타임라인
├─ learning-stage.ts          ← 7단계 학습 시스템 (글쓰기 진화)
├─ writing-mastery.ts         ← DB 연동 학습 파라미터
├─ memory-store.ts            ← RAG 벡터 저장/검색 (OpenAI)
├─ language-detector.ts       ← 언어 감지 (한/영/일)
├─ ai-config.ts               ← 모델 설정 (Opus/Sonnet/Haiku)
├─ visual-dna.ts              ← 사진 → 외형 묘사 (GPT-4o)
└─ universe/
   ├─ world-state.ts          ← 세계 상태 스냅샷/로깅
   ├─ npc-service.ts          ← NPC 상태/스케줄/상호작용
   └─ witness-service.ts      ← 목격자 기억 전파
```
